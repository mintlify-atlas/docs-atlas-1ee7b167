## Transform your Android device into a complete payment terminal

NetPOS is a comprehensive SaaS point-of-sale application designed for Android devices that enables merchants to accept multiple payment types, manage transactions, and process utility bills - all from a single platform.

## White-label ready

NetPOS is built with white-labeling support, allowing financial institutions and payment service providers to customize the application with their own branding. The platform currently supports multiple banking partners including:

- Zenith Bank
- Wema Bank  
- Providus Bank
- Unity Bank
- Heritage Bank
- And more

<Note>
  Each white-label flavor can be configured with custom branding, payment flows, and feature sets to match your institution's requirements.
</Note>

## Built for reliability

NetPOS is engineered for mission-critical payment processing:

- **Offline transaction storage** - Transactions are stored locally and synced when connectivity returns
- **Automatic retry logic** - Failed transaction uploads are automatically retried
- **Receipt printing** - Generate customer and merchant copies with built-in printer support
- **Security** - Encrypted PIN handling, secure key storage, and JWT authentication
- **Firebase integration** - Real-time push notifications for payment events
